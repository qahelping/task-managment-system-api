# Стили Automation Lab

Стили разбиты по страницам и подключаются только там, где нужны.

| Файл | Подключён в | Назначение |
|------|--------------|------------|
| `home.css` | `AutomationLabHomePage.tsx` | Секции, сетка фичей, карточки категорий, ссылки |
| `cards.css` | `CardsPage.tsx` | Банковские карточки (сетка, тиры, бейджи, адаптив) |

Базовые стили (layout, header, контейнер) по-прежнему берутся из `indigo-styles.css` и `aurora.css` через `AutomationLabLayout`.
